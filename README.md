# Gradio Beginner Tutorial

Welcome to the Gradio Beginner Tutorial! This course is designed for absolute beginners, guiding you step-by-step to build interactive apps using Gradio. Each lesson is simple, friendly, and builds on the previous one.

## Course Structure

Each lesson includes:
- A detailed tutorial ([lesson.md](lesson.md))
- A simple Gradio app ([app.py](app.py))

## How to Use This Course
1. Start with Lesson 1 and follow the tutorials in order.
2. Read the lesson.md file for explanations and analogies.
3. Run the app.py file to see the Gradio app in action.
4. Try the exercises and build your own apps!

## Getting Started

1. Make sure you have Python installed.
2. Install Gradio:
	```bash
	pip install gradio
	```
3. Go to any lesson folder and run the app:
	```bash
	python app.py
	```

## Who Is This For?
This course is for anyone who wants to learn Gradio from scratch. No programming experience needed!
